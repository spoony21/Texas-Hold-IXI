# Texas Hold'IXI

A multiplayer Texas Hold'em poker game built on the Spixi / Ixian network.

## How to Play

- 2–6 players, standard Texas Hold'em rules
- Blinds: 10 / 20, starting stack: 1000 chips
- Betting rounds: Pre-Flop → Flop → Turn → River → Showdown
- Actions: Fold, Check/Call, Raise, All-In

## Hand Rankings

Royal Flush > Straight Flush > Four of a Kind > Full House > Flush > Straight > Three of a Kind > Two Pair > One Pair > High Card

## Requirements

Requires the Spixi app. Open the game from within a Spixi conversation to start a session with other players.
